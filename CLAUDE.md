# Project: Personal Website

## Overview

A personal portfolio website to showcase the owner's profile, skills, work experience, and featured projects. The site also serves as a contact point for recruiters and potential collaborators.

## Repository

- GitHub: https://github.com/baohng/personal-website.git
- Default branch: `main`

## Status

Project is in the initial setup phase. Tech stack and architecture have not yet been decided.

## Conventions

- Communication with the maintainer happens in Vietnamese.
- All source code, comments, documentation, commit messages, and identifiers must be written in English.

## Repository Layout

- `README.md` — public-facing project description.
- `CLAUDE.md` — this file; project context for AI assistants.
- `plan/` — local-only directory for implementation plans and design notes. Ignored by git (see `.gitignore`); contents must not be pushed to the remote.
